# Radhika Gulati — Portfolio

A premium single-page portfolio website for an Economist & Data Scientist.

## 🚀 Quick Start

Simply open `index.html` in your browser — no build tools or dependencies needed.

```bash
# Or use a local server for development
npx serve .
```

## 📁 Structure

```
├── index.html                    # Main page
├── css/style.css                # Design system + all styles
├── js/main.js                   # Animations, scroll effects, canvas
├── assets/
│   └── Radhika G Master CV.pdf  # Downloadable CV
└── README.md                    # This file
```

## 🎨 Design

- **Typography**: Playfair Display (headings), Inter (body), Space Mono (stats)
- **Colors**: Navy/near-black dark sections, warm off-white light sections, copper accents
- **Animations**: Scroll-triggered reveals, animated stat counters, canvas scatter plot
- **Responsive**: Fully responsive across mobile, tablet, and desktop

## 🌐 Deployment (GitHub Pages)

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, root directory
4. (Optional) Add custom domain in the Custom Domain field
